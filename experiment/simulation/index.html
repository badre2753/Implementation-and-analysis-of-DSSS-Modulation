Connection Guide

  1. Connect Binary Sequence output (green) to NRZ Encoder input (red)
  2. Connect NRZ Encoder output to XOR Gate (top input)
  3. Connect PN Generator output to XOR Gate (bottom input)
  4. Connect XOR Gate output to BPSK Modulator (data input)
  5. Connect Carrier Generator output to BPSK Modulator (carrier input)
  6. Connect BPSK Modulator output to Output Signal input

Tip: Click on green output connectors first, then red input connectors to make connections.
